A high-performance one-page showcase website built for a craftsman specializing in interior and exterior renovation (wood decking, joinery, finishing work).

This project was developed for a client with the goal of strengthening their digital identity — presenting their expertise, conversion-focused web experience.

---

## Tech Stack

No heavy framework (React, Angular, etc.) was used in order to keep the site lightweight and maintainable.

- **HTML5** — Semantic structure optimized for SEO
- **Tailwind CSS** — Utility-first styling with a responsive, minimal footprint
- **Vanilla JavaScript** — Smooth interactions (mobile menu, scroll animations) with no runtime overhead
- **Vercel** — Continuous deployment and custom domain management

---

## Key Features

- **Lightweight performance** — No JS framework, targeting a near-perfect Google PageSpeed score
- **Mobile first** — Fully responsive layout optimized for smartphones
- **SEO ready** — Semantic markup (H1–H4), ALT attributes and meta tags configured for local search
- **Premium design** — Brushed-bronze textures, backdrop blur effects and elegant typography
